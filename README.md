# Estructura-de-datos-tecnm-3-semestr-
Para el archivo llamada (alumnos.py) los comentarios a dar serian:
1.-Visualización inteligente: Es excelente que no imprima toda la matriz; el "zoom" alrededor de la selección facilita la lectura.
2.-Punto débil: Si el usuario ingresa una letra en lugar de un número, el programa falla (falta un try/except).
3.-Medición de tiempo: El cronómetro incluye el tiempo que tardas en escribir; para ser exactos, debería empezar después de los input.
4.-Memoria: La advertencia de "mucha memoria" es innecesaria para este tamaño (3,000 datos), pero es un buen recordatorio para escalas mayores.
